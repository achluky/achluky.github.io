## ğŸš€ Getting Started

### Prerequisites
- Git
- Ruby and Bundler (for local Jekyll development)
- A GitHub account

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/achluky/achluky.github.io.git
cd achluky.github.io
```

2. Install Jekyll (if not already installed):
```bash
gem install bundler jekyll
```

3. Create a Gemfile:
```ruby
source 'https://rubygems.org'
gem 'github-pages', group: :jekyll_plugins
```

4. Install dependencies:
```bash
bundle install
```

5. Serve the site locally:
```bash
bundle exec jekyll serve
```

6. Open your browser and visit `http://localhost:4000`

### GitHub Pages Deployment

This site is automatically deployed to GitHub Pages when you push to the main branch. No additional configuration is required.
